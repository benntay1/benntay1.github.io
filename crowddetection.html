Real Time Crowd Detection

This project is being done as part of my machine learning curriculum at NUS.

Module code: CS3244 Machine Learning
Techniques: Deep Learning, Computer Vision, Convolutional Neural Networks

This is an ongoing project where we make use of convolutional neural networks to conduct crowd counting on a static image. We make use of a CNN-based density estimation approach to generate a density map to build an end-to-end crowd counting model.

Datasets used (for training and evaluation): JHU++ large scale crowd counting dataset, ShanghaiTech A and B dataset. Data is consolidated into a common format and store within HDF5 files.

After loading in the datasets, we first implemented two neural networks, namely MCNN and CSRNet, which are both structurally different from each other (the former being a multi-column NN and the latter a single-column NN).

Our team aims to evaluate and make improvements to the chosen models through the creation of masks which blocks out potential noise in our raw images to generate more accurate density maps for estimation.
